# Computer_Science_project
Project name: Comparitive Analysis of Machine Learning Models for URL Detection
Description : This project provides binary classification of machine learning algorithms to detect the unsafe and safe uniform resource locator (URL). 

  1.Dataset Size: 49,750 samples
  2.Features: 104 ( 4 Lexical & 100 TF-IDF)

Installation :
  1. Prerequisite libraries:
    pip install pandas numpy scikit-learn matplotlib seaborn
  2. Rename dataset file as urls.csv and place it project root directory
